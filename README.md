User Stories
-----------
```
As a User
So that I can use the Bookmark Manager
I want to sign up to the site

As a User
So that I can access the Bookmark Manager
I want to sign in to the site

As a User
So that I can add a bookmark
I want to store a link and a title

As a User
So that I can group links
I want to tag a link

As a User
So that I can change groups
I want to modify a tag

As a User
So that I can remove outdated tags
I want to delete a tag

As a user
So that I can manage my tags
I want to see all my tags

As a User
So that I can view my stored link
I want to retrieve a list of all saved links

As a User
So that I can see the last links I added
I want to sort my links in chronological order

As a User
So that I can view similar links
I want to retrieve a list of links with the same tag

As a User
So that I can remove outdated links
I want to delete a link

As a User
So that I can modify errors
I want to update a link

As a User
So that I can make sure my data is private
I want to sign out
```
